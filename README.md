# Run Student !

Aujourd'hui, c'est contrôle ! Malheureusement, vous n'avez pas révisé.
Copiez sur vos camarades pour tenter de sauver votre note, mais attention, à ne pas vous faire prendre !
3 bonnes réponses, et c'est le 20/20 qui vous attend !





<!-- ABOUT THE PROJECT -->
## A propos
Ce jeu a été réalisé en 3ème année de BUT informatique au Puy-en-Velay.

Il a été réalisé sur Unity.



<!-- GETTING STARTED -->
## Téléchargement

## Contrôles:

ZQSD / flèches directionnelles : Bouger
E :  Copié / Se cacher
A:  S'asseoir
C:  Changer de caméra
